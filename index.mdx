## Core Features

StreamLine Logistics is a production-ready microservices architecture built with Spring Boot, demonstrating modern cloud-native patterns for logistics and e-commerce applications.

<CardGroup cols={2}>
  <Card title="Microservices Architecture" icon="grid" href="/architecture">
    Independent services for orders, inventory, and tracking with clean separation of concerns
  </Card>
  <Card title="Service Discovery" icon="compass" href="/infrastructure/eureka-server">
    Netflix Eureka for automatic service registration and discovery
  </Card>
  <Card title="API Gateway" icon="gateway" href="/infrastructure/api-gateway">
    Unified entry point for all client requests with routing and load balancing
  </Card>
  <Card title="Hexagonal Architecture" icon="hexagon" href="/microservices/overview">
    Ports and adapters pattern for clean, testable, and maintainable code
  </Card>
</CardGroup>
